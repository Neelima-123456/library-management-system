# Library Management System

## Description
Library Management System is a web-based application that helps manage library operations such as adding books, issuing books, returning books, and maintaining student records.

## Modules
- Admin
- Student

## Features
- Book Management
- Issue and Return Books
- Student Management
- Search Books

## Technologies Used
- Java
- MySQL
- HTML
- CSS
- JSP & Servlets

## Project Status
Project documentation completed. Development in progress.
